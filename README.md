## üîß Quick Start

### 1. Install PHP

**macOS:**
```bash
brew install php
php -v
```

**Ubuntu/Debian:**
```bash
sudo apt-get update
sudo apt-get install php php-cli php-mysql
php -v
```

**Windows:**
- Download from [php.net](https://www.php.net)
- Or use XAMPP/WAMP for bundled setup

### 2. Run Examples

```bash
# Navigate to a topic
cd 01-basics/

# Run a PHP file
php 3-hello-world.php

# Or use the built-in PHP server for web files
php -S localhost:8000
# Then visit http://localhost:8000
```
